# Quant Dashboard
AI-powered quantitative trading simulator built with React + Recharts.

## Setup
git clone https://github.com/Harsh-Naruka/Quant-Dashboard.git
cd Quant-Dashboard
npm install
npm run dev

## Features
- Compare 6 AI models (SVM, LSTM, GPT-4, etc.)
- Adjust transaction costs and volatility
- Live equity curve simulation
- Sharpe ratio, max drawdown, and alpha tracking
