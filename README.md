# AI Quant Trading Simulator

AI Quant Trading Simulator is an interactive React-based dashboard that simulates and compares the performance of multiple machine learning models in algorithmic trading. It enables users to analyze how models like SVM, Random Forest, LSTM, Transformer (BERT), Hybrid CNN-LSTM, and GPT-4 perform under different market conditions using key financial metrics and visualizations.

## 🚀 Features

* 📊 Model comparison using Accuracy, Annualized Return, Sharpe Ratio, and Max Drawdown
* 📈 12-month equity curve simulation with stochastic modeling
* ⚙️ Adjustable transaction costs and volatility multiplier
* 📉 Benchmark comparison against passive portfolio
* 🎯 Alpha calculation and portfolio performance insights
* 🎨 Clean, modern dark-themed UI with interactive charts

## 🧠 Tech Stack

* React (Hooks & Functional Components)
* Recharts (Data Visualization)
* JavaScript (Simulation Logic)

## 📌 Use Cases

* Quantitative finance experiments
* Machine learning model comparison
* Portfolio strategy simulation
* Academic research visualization

## 💡 Tagline

Simulating AI-driven trading strategies with real-time performance analytics.

## 📖 Explanation

This project demonstrates how advanced machine learning models can be applied in quantitative finance to evaluate trading strategies.
It highlights the importance of balancing return and risk using metrics like Sharpe Ratio and Maximum Drawdown.
Users can dynamically adjust transaction costs to observe how trading friction impacts profitability.
The volatility multiplier allows simulation of different market conditions such as stable or highly volatile environments.
Each model behaves differently under changing parameters, showcasing strengths and weaknesses.
The dashboard provides real-time visual feedback, making complex financial concepts easier to understand.
It is especially useful for students and developers exploring AI in finance.
Overall, it bridges the gap between theoretical models and practical trading insights.

## 🧪 Example Scenario

For example, selecting the GPT-4 Fine-tuned model shows higher returns compared to traditional models.
If transaction costs are increased, the net return decreases, demonstrating cost sensitivity.
By increasing volatility, the Sharpe Ratio drops, indicating higher risk exposure.
Switching to Random Forest provides lower returns but more stability in certain cases.
The benchmark portfolio helps compare whether the AI model is actually outperforming the market.
The equity curve visually represents how the portfolio grows over time month by month.
Users can experiment with different configurations to find optimal strategies.
This makes the simulator a powerful tool for understanding real-world trading behavior.
